BTCPay Server

BTCPay Server es el estándar para negocios que quieren aceptar Bitcoin de forma soberana — sin comisiones de terceros, sin custodiar tus fondos otros. Es software libre y open source.

La comunidad ₿olivia.ORG opera su propio servidor BTCPay para que los negocios bolivianos puedan procesar pagos Bitcoin con solo escanear un QR.

Billeteras Frías

Las billeteras de hardware (o "frías") son el estándar de oro para guardar Bitcoin a largo plazo. Mantienen tus claves privadas fuera de internet — a salvo de hackers.

Recomendamos y vendemos (o gestionamos el acceso a) las principales billeteras disponibles para bolivianos:

Trezor Safe 3 / Safe 5
Open source, fabricado en la República Checa. El más usado por la comunidad boliviana.
Consultar disponibilidad →
Jade de Blockstream
Bitcoin-only, open source, sin chip seguro propietario. Excelente relación calidad-precio.
Consultar disponibilidad →
Nota: Los precios se cobran en Bitcoin. Escríbenos para gestionar tu compra.
